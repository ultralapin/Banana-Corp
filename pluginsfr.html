Plugins du serveur


Le serveur tourne avec Oxide 2 et plusieurs plugins viennent l'enrichir, cette liste peut changer.

  • SignArtists: Ce plugin vous permet d'importer une image sur un cadre ou une pancarte. Une fois face au cadre tapez "/sil url" (remplacez url par l'adresse EXACTE de l'image qui doit se finir par.png ou .jpg). Si vous voulez mettre vos images il faudra les heberger avant (imgur est simple pour ça).
  • RemoverTool: Ce plugin vous permet, en étant autorisé sur une armoire, de détruire tout ce qui est dans son périmètre avec un "/remove" puis un clic sur l'objet à enlever (mur, porte, pancarte).
    La destruction coute un peu de ressources mais vous en récuperez aussi.
    Très pratique pour étendre ou changer votre maison, PROTEGEZ BIEN VOTRE ARMOIRE !
  • CupboardRadius: Ce plugin (passif) étend la zone de contrôle d'une armoire à 35m.
  • ZLevelsRemastered: Ce plugin (passif) vous fait monter de niveau en ramassant des ressources, plus vous êtes haut plus vous ramassez !
  • DeathNotes: Ce plugin (passif) affiche les morts/kill du serveur (en anglais uniquement).
  • NoDecay: Ce plugin (passif) desactive le decay (usure).
  • NightLantern: Ce plugin (passif) allume les lanternes/lampes de plafond/lampe citrouilles/lampes boite de thon à la nuit tombé et ce sans besoin de carburant.
  • CorpseDuration: Ce plugin (passif) rallonge la durée avant que votre corp ne disparaisse à 10 minutes. Attention, malgré cela les corps disparaissent souvent sans raison dans rust pour le moment.
  • AutoDoors: Ce plugin (passif) referme vos portes tout seul au bout de 8 secondes.
