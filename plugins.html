Server's plugins


The server is running with Oxide 2 and some plugins came to the party, this list can change over time.

  • SignArtists: This plugin allow you to importa picture on a frame. Face the frame and type "/sil url" (replace url by the EXACT adress of the picture, it must end by .png or .jpg). If you want to put your pictures you'll have to host them before (imgur is great).
  • RemoverTool: This plugin allow you, when authorize on a cupboard, to destroy everything in it perimeter with a "/remove" then a clic on the object you want to remove (wall, door, frame).
    The destruction cost you ressources but you'll have some back.
    Very nice if you want to change or expend your house, PROTECT YOUR CUPBOARD !
  • CupboardRadius: This plugin (passive) extend the contr√¥le zone of a cupboard to 35m.
  • ZLevelsRemastered:This plugin (passive) make you level up will harvesting ressources, the higher you are, the higher you get !
  • DeathNotes: This plugin (passive) display the death/kill of the server.
  • NoDecay: This plugin (passive) remove the decay.
  • NightLantern: This plugin (passive) light on the lanterns/ceiling lights/pumpkin lights/Tuna cans lamps when the night fall. No fuel is required.
  • CorpseDuration: This plugin (passive) modify the length of which a player's corpse remains active in the world to 10 minutes. Attention, even with this corpse often disappear in Rust.
  • AutoDoors: This plugin (passive) close the doors behind you in 8 seconds.
